Part 3. Conway's Game of Life

This interactive simulation is a playground for cellular life!

Try clicking and dragging your mouse around on the board to create new life! Controls for the simulation are at the top of the panel, and rules at the bottom.

Conway's Game

The default ruleset (Conway's game) reads: when a cell's neighbour count is less than 4 and greater than 2 (exactly three) make it alive. If less than 2 or greater than 3, cell dies

Maze Generator

With some modifications to the rules, these cells can create a maze-like pattern. Experiment and try to find it yourself

Only one rule needs to be changed. Use the reset button in the bottom right to reset rules

The last rule needs to be changed by one to make a maze.

Cave Generator

Some video games use cellular automata to create cave-like levels. Experiment to find this ruleset.

This pattern only needs two of the four rules. Place and X in the first and last rules and set both boolean conditions to OR

The integers in the second and third rule should match.
